# Screenshots

Captura usada no `README.md`.

Padrão de captura recomendado:

- Resolução: 1440×900 ou 1600×1000 (proporção wide).
- Tema: claro (a aplicação só tem light mode).
- Idioma do navegador: pt-BR.
- Janela limpa: sem extensões visíveis.
- Exportar como PNG.
